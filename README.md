# Saraswati Library Management System

A web-based library management system built with **Python**, **Flask**, and **SQLite**.
Features
* **User Authentication:** Secure login and registration.
* **Book Management:** Add, view, and search for books.
* **Dashboard:** A clean interface for librarians and students.
 How to run
1. Clone the repository: `git clone [YOUR_URL]`
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `python app.py`
